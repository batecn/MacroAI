MacroAI 使用者手册

一、简介

MacroAI 是一款 Windows 宏自动化工具。您可以使用他人制作好的自动化脚本，让软件自动完成鼠标点击、键盘输入、图像识别等一系列操作。

适用场景： 游戏辅助 · 办公自动化 · 软件测试 · 运维管理 · RPA

二、快速上手

1. 导入项目

从设计者那里获得项目文件（.zip 格式）后：

  1. 点击菜单 项目 → 导入（或按 Ctrl+I）
  2. 选择 zip 文件
  3. 如果设计者标记了分发包，系统会自动进入运行模式

什么是分发包？设计者导出的特殊项目包，导入后自动进入运行模式且无法切换至设计模式，您只需关注运行所需的配置即可。

2. 认识运行模式界面

导入分发包后，您将看到简洁的运行模式界面，主要分为三个区域：

┌── ① 项目信息（可折叠） ───────────────────────┐
│  📁 项目名称                                   │
│  设计分辨率：1920×1080   坐标策略：等比缩放     │
│  ────────────────────────────────────────────  │
│  项目描述：自动识别游戏界面，匹配相同图案       │
│  并自动点击消除。可自定义头像和昵称显示。       │
└──────────────────────────────────────────────────┘

┌── ② 区域校准（始终可见） ───────────────────┐
│  🖥 运行区域：── 未校准 ──  [ 校准运行区域 ]  │
│  🖥 设计区域：1920×1080（参考）               │
│  坐标策略：等比缩放（自动适配）               │
└──────────────────────────────────────────────────┘

┌── ③ 个性化配置（可折叠） ───────────────────┐
│  ⚠️ 昵称：使用默认值                          │
│  ✅ 头像：已设置                              │
│  ❌ 签名档：未设置                            │
└──────────────────────────────────────────────────┘

工具栏提供运行/停止、后台/前台切换、日志级别控制等基本功能。

3. 第一步：校准运行区域

校准是使用脚本的第一步，确保脚本的操作位置与您的屏幕匹配：

  1. 点击区域校准卡片上的 [ 校准运行区域 ] 按钮
  2. 按照屏幕指示框选脚本运行的窗口或区域
  3. 校准完成后，卡片上会显示运行区域的具体尺寸和位置

如果设计区域是全屏，系统会自动适配当前屏幕，无需手动框选。

4. 第二步：填写个性化配置

如果设计者设置了用户配置变量（如昵称、头像等），在个性化配置区填写：

  • 文本类型：在输入框中输入您的值
  • 图像类型：点击 [截图] 或 [选择文件] 上传您的图片

每项配置左侧的状态图标表示：

图标 含义
✅ 您已设置自己的值
⚠️ 未修改，使用设计者的默认值
❌ 未设置（仅无默认值的必填项）

所有修改会自动保存。

5. 运行脚本

所有配置完成后，点击 ▶ 运行（或按 F5）执行脚本。点击 ■ 停止（或按 Shift+F5）随时终止。

运行过程中，底部的日志面板会自动展开，显示执行进度和结果。

三、配置完整度指示器

工具栏右侧的指示器帮助您了解当前配置状态：

状态 含义 操作
🟢 就绪，可以运行 所有配置完成 + 区域已校准 直接运行
🟡 X项使用默认值 有配置未个性化但区域已校准 可运行，建议完善
🔴 区域待校准 运行区域尚未校准 需先校准再运行

点击指示器可展开详细配置清单，查看具体哪些项需要关注，点击 [跳转] 快速定位。

四、日志面板

运行日志显示在界面底部的可折叠面板中：

  • 空闲时：面板收起，底部只显示运行按钮
  • 运行中：面板自动展开，实时显示日志
  • 运行结束：保留最后状态，可手动收起

工具栏的 🐞 调试 下拉菜单控制日志详细程度：

级别 显示内容
简洁（默认） 关键操作提示 + 错误信息
追踪 简洁内容 + 每个步骤的执行过程
调试 追踪内容 + 详细技术信息

五、浮动控制面板

运行脚本时自动弹出浮动控制面板（始终置顶）：

  • 暂停/继续：临时停止或恢复执行
  • 自定义动作按钮：设计者定义的快捷操作（如"暂停后手动调整窗口"）
  • 背景音乐控制：如果脚本包含音乐，显示暂停/继续/静音按钮
  • 关闭面板：停止脚本执行

六、常见问题

校准区域时框选错了怎么办？

重新点击 [ 校准运行区域 ] 按钮再次框选即可。

个性化配置填错了怎么恢复？

文本类型点击 [恢复默认] 即可还原为设计者的示例值；图像类型点击 [清除] 后重新选择。

状态指示器一直显示🔴？

说明运行区域尚未校准。请完成校准操作后再运行。

运行过程中脚本点击位置不对？

可能是运行区域校准不准确，请重新校准。如果问题持续，联系脚本的设计者。

如何切换后台/前台模式？

  • 后台模式（默认）：在后台模拟操作，不影响您正常使用电脑
  • 前台模式：直接控制鼠标键盘，会抢占控制权

后台模式无效时可切换为前台模式。
